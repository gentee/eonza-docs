# Help

Here you can send the developers any comments and suggestions, as well as get more information about the Eonza program.

## Feedback

Send us any feature requests, comments or bug reports. Your feedback will help to improve the Eonza program and make it more user-friendly. You can also send an email or create an issue on [GitHub](https://github.com/gentee/eonza/issues).
