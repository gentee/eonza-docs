# VeraCrypt

The **VeraCrypt** package contains scripts for working with the [VeraCrypt](https://veracrypt.fr/) cryptographic software. VeraCrypt is free, open source, cross-platform disk encryption software.

%html.veracrypt%

## Settings

**Application File**  
Specify the full path to the VeraCrypt application file.

```txt
C:\Portable\VeraCrypt\VeraCrypt.exe
/usr/bin/veracrypt
```
