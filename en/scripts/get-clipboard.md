# Get Clipboard

The **Get Clipboard** command saves the contents of the clipboard to a variable.

%html.get-clipboard%

**Variable Name**  
Specify the name of the variable to which the contents of the clipboard will be assigned. The command can retrieve only the text from the clipboard.

**Clear Clipboard**  
Check this checkbox if you want to clear the clipboard after receiving its contents.
