# Foreach Line in File

**v1.10.0.**  
The **Foreach Line in File** command executes nested commands for each line in the specified text file.

%html.foreach-line-file%

**File name**.  
Specify the full or relative path to the file where you want to read each line and execute nested commands.

**Variable Name**  
Specify the name of the variable to which the next line for processing will be written. You can use this variable in nested scripts.
