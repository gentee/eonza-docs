# Notification

The **Notification** command sends a notification to the Eonza program. You can see the received notifications in the [Notifications](/docs/notifications.html) tab.

%html.notification%

**Text**.  
Specify the text of the notification to be sent. The text may contain HTML tags.

``` txt
Hello, #name#,
The test task has been <strong>successfully</strong> finished.
```
