# Call Function

The **Call function** command calls a function that has been defined with the [Function](function.html) command.

%html.call-function%

**Function Name**  
Specify the name of the function to be called.

## Parameters

You can assign values to variables that are used in the called function.

**Variable Name**  
Specify the name of the variable to which you want to assign a value.

**Value**  
Specify the value to assign. It can be multiline.
