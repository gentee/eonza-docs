# Download File

The **Download File** command downloads the file at the specified URL and saves it in the specified directory.

%html.download-file%

**URL**  
Specify the address of the file you want to download.

``` txt
https://#mydomain#/archive/myapp.zip
```

**Destination Folder**.  
Specify the directory where you want to save the downloaded file.

**New Filename**  
By default, the file is saved with the name specified in the URL. But you can specify a different name for the saved file in this field.
