# Unpack ZIP

The **Unpack ZIP** command unpacks the zip archive to the specified directory. The command can unpack individual files according to the specified patterns.

%html.unpack-zip%

**Filename**.  
Specify the full or relative name of the *.zip* archive.

**Destination Folder**  
Specify the directory where you want to unpack the archive file. By default, all files and subdirectories will be unpacked.

You can specify patterns for files to be packed and for files and directories to be skipped.
