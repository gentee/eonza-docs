# Copy To Clipboard

The **Copy To Clipboard** command copies the specified text to the clipboard.

%html.copy-to-clipboard%

**Text**.  
Specify the text that you want to copy to the clipboard. If no text is specified, the clipboard will be cleared.

``` txt
Name: #name#
Company: #company#
```
