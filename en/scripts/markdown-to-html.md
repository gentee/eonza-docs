# Markdown To HTML

**v1.15.0**  
The **Markdown to HTML** command converts Markdown text to HTML format.

%html.markdown-to-html%

**Variable Name**  
Specify the name of the variable that contains the text in Markdown format.

**Result variable**  
Specify the name of the variable in which the resulting HTML text will be written. If no variable name is specified, the result will be assigned to the original variable.
