# SQL Close

**1.12.0**  
The **SQL Close** command closes the connection to the SQL server.

%html.sql-close%

**SQL Connection**  
Provide a variable name with the SQL connection identifier. This variable must be defined in the [SQL Connection](sql-connection.html) command.
