# Foreach Line

The **Foreach Line** command executes nested commands for each line in the specified text. This command is convenient to use when you want to perform some action for each line in a certain set.

%html.foreach-line%

**Text**  
Specify the source text that contains the lines to be processed.

**Variable Name**  
Specify the name of the variable in which the next line to be processed will be written. You can use this variable in nested scripts.
