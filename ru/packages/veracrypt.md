# VeraCrypt

Пакет **VeraCrypt** содержит скрипты для работы с криптографической программой [VeraCrypt](https://veracrypt.fr/). VeraCrypt - это бесплатное кроссплатформенное программное обеспечение с открытым исходным кодом для работы с зашифрованными томами.

%html.veracrypt%

## Настройки

**Файл приложения**  
Укажите полный путь к файлу приложения VeraCrypt.

```txt
C:\Portable\VeraCrypt\VeraCrypt.exe
/usr/bin/veracrypt
```
