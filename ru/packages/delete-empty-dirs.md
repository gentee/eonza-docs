# Удалить пустые директории

Скрипт **Удалить пустые директории** ищет пустые директории и удаляет их. При желании можно выводить список пустых директорий и отмечать вручную директории для удаления.

%html.delete-empty-dirs%

**Путь**  
Укажите директорию, в которой нужно искать и удалять пустые директории.

**Маска или регулярное выражение**  
**Исключать по маске или рег.выраж.**  
Эти параметры описаны в команде [Цикл для каждого файла](/ru/scripts/foreach-file.html).

**Подтвердить удаление**  
Отметьте этот чекбокс, если вы хотите самостоятельно выбрать какие из найденных директорий следует удалить. В этом случае, вы увидите список найденных пустых директорий и скрипт удалит только отмеченные вами директории.

%html.delete-empty-dirs-confirm%
