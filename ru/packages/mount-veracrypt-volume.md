# Монтировать том VeraCrypt

Команда **Монтировать том VeraCrypt** монтирует указанный зашифрованный файл с помощью программы VeraCrypt. Для работы скрипта необходимо, чтобы был указан путь к файлу приложения VeraCrypt в настройках пакета.

%html.mount-veracrypt-volume%

**Тoм VeraCrypt**  
Укажите полный путь к зашифрованному файлу VeraCrypt, который необходимо смонтировать.

**Пароль**  
Укажите пароль, который использовался для шифрования монтируемого тома. Не указывайте пароль в явном виде. Просите ввести пароль в процессе работы вызывающего скрипта или используйте секретное хранилище в Pro версии.

**Буква диска/Директория монтирования**  

* **Windows**. Укажите букву диска, на котором будет смонтирован том VeraCrypt. Если не указана, то том будет смонтирован на ближайшей свободной букве диска.
* **Linux**. Укажите директорию, в которой будет смонтирован том VeraCrypt. Если не указана, то том будет смонтирован в директории */mnt*.

**Файл-ключ**  
Укажите файл-ключ, если он использовался при шифровании данных.

**Пароль sudo (для Linux)**  
При запуске на Linux необходимо указать sudo пароль. Не указывайте пароль в явном виде!

## Опциональные параметры

Вы можете указать дополнительные параметры для монтирования тома.

**pim**  
PIM идентификатор.

**slot**  
При использовании на Linux можно указать номер слот, на котором будет смонтирован том.
