# Найти одинаковые файлы

Скрипт **Найти одинаковые файлы** ищет одинаковые файлы в указанной директории и всех поддиректориях. Он вызывает вложенные команды для каждого множества найденных одинаковых файлов. Файлы считаются одинаковыми, если у них равен размер и **MD5** хэш. 

%html.find-duplicate-files%

**Путь**  
Укажите директорию, в которой нужно искать одинаковые файлы. Одинаковые файлы будут искаться во всех поддиректориях.

**Маска или регулярное выражение**  
**Исключать по маске или рег.выраж.**  
Эти параметры описаны в команде [Цикл для каждого файла](/ru/scripts/foreach-file.html).

**Имя переменной**  
Укажите имя переменной, которой будет присвоен ассоциативный массив-объект со следующими полями.

* **size** - размер одинаковых файлов;
* **md5** - MD5 хэш одинаковых файлов;
* **list** - массив полных имен найденных одинаковых файлов.

Вложенные команды выполняются для каждого множества одинаковых файлов. Вы можете использовать эту переменную во вложенных командах для выполнения нужных действий.
