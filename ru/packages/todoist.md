# Todoist

**v1.23**  
Пакет **Todoist** содержит скрипты для работы с сервисом планирования [Todoist](https://todoist.com/). Все скрипты используют REST API сервера Todoist для управления задачами. Для работы все скриптов пакета необходимо указать API Токен в настройках пакета.

%html.todoist%

## Настройки

**API URL**  
Укажите URL, которому будут отправляться REST API запросы. По умолчанию он равен *https://api.todoist.com/rest/v1/*.

**Токен**  
Так как сервис Todoist требует аутентификации для доступа к вашим задачам, то следует указать токен аккаунта. Его можно найти в меню Настройки вашего аккаунта, в разделе Интеграции. Никому не показывайте и не передавайте этот API токен. Для безопасной работы рекомендуется использовать Eonza Pro версию и хранить токен в зашифрованном хранилище, а здесь указать только имя соответствующего значения.
