# Создать Todoist задачу

Команда **Создать Todoist задачу** создаёт указанную задачу в сервисе Todoist. Для работы скрипта необходимо, чтобы был указан API токен в настройках пакета.

%html.create-todoist-task%

**Содержание**  
Наименование добавляемой задачи. Оно будет отображаться в списке задач Todoist.

**Срок выполнения**  
Укажите срок выполнения задачи. Вы можете указать конкретную дату в формате *YYYY-MM-DD* или используя [описания](https://todoist.com/help/articles/due-dates-and-times), которые поддерживает Todoist. Например, *tomorrow at 12:00*.

## Опциональные параметры

Вы можете указать дополнительные параметры для создаваемой задачи.

**desc**  
Дополнительное описание задачи.

**project**  
Идентификатор проекта, в которые следует добавить задачу.

**section**  
Идентификатор секции, в которую нужно добавить задачу.

**priority**  
Приоритет задачи.
