# Размонтировать том VeraCrypt

Команда **Размонтировать том VeraCrypt** размонтирует VeraCrypt том. 

%html.dismount-veracrypt-volume%

**Буква диска/Директория монтирования**  

* **Windows**. Укажите букву диска, которую нужно размонтировать. 
* **Linux**. Укажите директорию, которую нужно размонтировать. 

Если параметр не указан, то будут размонтированы все VeraCrypt тома.

**Пароль sudo (для Linux)**  
При запуске на Linux необходимо указать sudo пароль. Не указывайте пароль в явном виде! Просите ввести пароль в процессе работы вызывающего скрипта или используйте секретное хранилище в Pro версии.
