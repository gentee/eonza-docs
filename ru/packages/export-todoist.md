# Экспорт Todoist

Бесплатный аккаунт Todoist не имеет возможностей для экспорта существующих проектов и задач. Как в этом случае делать резервные копии содержимого Todoist? Скрипт **Экспорт Todoist** получает ваши проекты, секции, задачи, комментарии и метки в JSON формате и упаковывает их в zip файл. Для работы скрипта необходимо, чтобы был указан API токен в настройках пакета.

%html.export-todoist%

**Путь**  
Укажите директорию, куда будет сохранён архивный файл.

**Имя архива (.zip)**  
Определите имя создаваемого .zip файла.

Архив с резервной копией ваших данных будет содержать следующие файлы:

* comments.json - комментарии;
* labels.json - метки;
* projects.json - проекты;
* sections.json - секции;
* tasks.json - задачи.
