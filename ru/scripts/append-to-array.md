# Добавить к массиву

**v1.21.0**  
Команда **Добавить к массиву** добавляет элемент к переменной объекту. Объект должен быть массивом.
Если указанная переменная не существует, то она будет создана.

%html.append-to-array%

**Имя переменной**  
Укажите имя переменной, которая является массивом или которую вы хотите создать. Вы можете указывать
имена массивов в существующих объектах.

```txt
mylist
options.list
my[0].dir.sub
```

**Значение**  
Укажите добавляемое значение. Оно может быть многострочным. По умолчанию, к массиву будет добавлен объект с этой строкой. Также, вы можете указать имя другого объекта. Скрипт проверяет, если существует объект с указанным именем, то в этом случае, к массиву будет добавлен этот объект.

**Заменить значения переменных**  
Значения этого параметра описаны в команде [Установить переменную](set-variable.html).
