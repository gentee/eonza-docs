# Элемент блок-схемы

Команда **Элемент блок-схемы** создаёт страницу для элемента блок-схемы. Команда может содержать другие скрипты, которые будут выполняться при переходе к этому элементу. Вы можете указать необходимый текст и список дальнейших действий, которые реализуют переход к другим элементам блок-схемы. Эта команда похожа на функцию и выполняет внутренние команды только при переходе к этому элементу. Чтобы запустить блок-схему нужно использовать команду [Блок-схема](flowchart.html).

%html.flowchart-element%

**Имя**  
Укажите имя элемента. Используйте это имя в вариантах действий других элементов. При выборе варианта с таким именем, пользователь перейдёт на этот элемент.

**Название**  
Заголовок страницы. Если не указан, то заголовок будет равен имени элемента.

**Текст**  
Текст страницы. Вы можете использовать текст в формате HTML или Markdown. 

```md
Current value: **#num#**
```

## Следующие действия

**Текст**  
Укажите текст ссылки для варианта. Если не указан, то будет использоваться заголовок элемента, который определён в следующем поле.

**Элемент блок-схемы**  
Укажите имя элемента блок-схемы, который выполнится при выборе данного варианта.
