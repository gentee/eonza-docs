# Распаковать TAR.GZ

Команда **Распаковать TAR.GZ** распаковывает tar.gz архив в указанную директорию. Команда может распаковывать отдельные файлы в соответствии с указанными шаблонами.

%html.unpack-targz%

**Имя файла**  
Укажите полное или относительное имя *.tar.gz* архива.

**Целевая папка**  
Укажите директорию, куда необходимо распаковать файл архива. По умолчанию, будут распакованы все файлы и поддиректории.

Вы можете указать шаблоны для файлов, которые необходимо распаковывать и для файлов и директорий, которые нужно пропускать.

**Маска или регулярное выражение**  
**Исключать по маске или рег.выраж.**  
Эти параметры описаны в команде [Цикл для каждого файла](foreach-file.html).
