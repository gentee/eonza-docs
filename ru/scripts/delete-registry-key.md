# Удалить ключ Реестра

**v1.14.0**  
Команда **Удалить ключ Реестра** удаляет указанный ключ Реестра Windows со всеми подключами.

%html.delete-registry-key%

**Корневой ключ**  
Укажите корневой ключ Реестра.

**Подключ**  
Укажите имя ключа Реестра, который должен быть удален. Указанный ключ будет удалён со всеми подключами.

``` txt
Software\My Application\Data
```

**Доступ**  
Доступ для ключей 32-битных или 64-битных приложений.

* *---* - доступ по умолчанию.
* *WOW64_32KEY* - доступ к ключам Реестра для 32-битных приложений.
* *WOW64_64KEY* - доступ к ключам Реестра для 64-битных приложений.
