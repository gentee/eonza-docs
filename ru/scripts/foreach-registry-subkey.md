# Для каждого подключа Реестра

**v1.14.0**  
Команда **Для каждого подключа Реестра** выполняет вложенные команды для каждого подключа в указанном ключе Реестра Windows.

%html.foreach-registry-subkey%

**Корневой ключ**  
Укажите корневой ключ Реестра.

**Подключ**  
Укажите имя подключа Реестра в котором нужно перебрать все подключи.

``` txt
Software\My Application
```

**Имя переменной**  
Укажите имя переменной, в которую будет записанo имя очередного подключа. Вы можете использовать эту переменную во вложенных командах.

**Доступ**  
Доступ для ключей 32-битных или 64-битных приложений.

* *---* - доступ по умолчанию.
* *WOW64_32KEY* - доступ к ключам Реестра для 32-битных приложений.
* *WOW64_64KEY* - доступ к ключам Реестра для 64-битных приложений.
