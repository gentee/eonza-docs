# Для каждой строки в файле

Команда **Для каждой строки в файле** выполняет вложенные команды для каждой строки в указанном текстовом файле.

%html.foreach-line-file%

**Имя файла**  
Укажите полный или относительный путь к файлу, в котором вы хотите прочитать каждую строку и выполнить вложенные команды.

**Имя переменной**  
Укажите имя переменной, в которую будет записана очередная строка для обработки. Вы можете использовать эту переменную во вложенных  скриптах.
