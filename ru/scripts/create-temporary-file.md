# Временный файл

Команда **Временный файл** создаёт временный файл со случайно сгенерированным именем. В дальнейшем, желательно  удалить созданный файл, если он стал не нужен.

%html.create-temporary-file%

**Текст**  
Укажите текст, который будет записан в созданный файл.

**Шаблон имени файла**  
По умолчанию создается имя файла, которое состоит из случайных цифр. Вы можете указать префикс, который будет добавлен в начале имени. Если нужно, чтобы случайное число было в середине имени файла, то укажите в нужном месте шаблон символ `*`.

```txt
my => my4604338
pref*.txt => pref1028002494.txt
```

**Результирующая переменная**  
Укажите имя переменной, в которую будет записано полное имя созданного файла.

## Опциональные параметры

**path**  
По умолчанию файл создаётся в системной временной директории. Вы можете указать любую директорию, где вы хотите создать временный файл.

```txt
path: /home/user/tmp
```
