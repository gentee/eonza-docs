# Шаблон DOCX

Команда **Шаблон DOCX** заменяет переменные в Docx шаблоне (Microsoft Word). Так как скрипт сам производит все замены в документе, то ему не требуется установленного MS Office. Если команда не заменяет какие-то переменные в шаблоне, то нужно отдельно скопировать их (*#varname#*) из обычного текстового редактора и вставить в документ. Это связано с тем, что
Microsoft Word может разбить имя переменной на невидимые фрагменты, которые скрипт не может обработать как одно целое.

%html.docx-template%

**Шаблон (.docx)**  
Укажите *docx* файл, которые содержит переменные вида *#varname#*. Данный скрипт заменит все переменные и
сохранит новый документ с указанным ниже именем.

**Файл вывода**  
Укажите полный или относительный путь к документу, который нужно создать. Если файл существует, то он будет перезаписан.
