# Получить ячейки Excel

Команда **Получить ячейки Excel** получает значения указанных ячеек в электронной таблице Excel и присваивает их переменным.

%html.get-excel-cells%

**Excel файл (*.xlsx)**  
Укажите полный или относительный путь к XLSX файлу, из которого вы хотите получить значения ячеек.

## Ячейки

Можно указать одну или несколько ячеек Excel, значения которых вы хотите получить.

**Лист**  
Укажите имя листа в электронной таблице. Если этот параметр не указан, то значения будут браться из первого листа.

**Ячейка**  
Укажите имя ячейки в электронной таблице. Например, *A10*, *C7*, *B1*.

**Имя переменной**  
Имя переменной, которой будет присвоено значение указанной ячейки Excel.
