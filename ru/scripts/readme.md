# Прочти меня

Команда **Прочти меня** показывает HTML, Markdown или преварительно отформатированный текст. Также эта команда может использоваться для описания возможностей скрипта, документирования и т.д.

%html.readme-1%

**Текст**  
Укажите текст, который необходимо показать пользователю. Если текст начинается с **&lt;**, то он будет показан как HTML. Если текст начинается с одного или несколько символов **#** и пробела, то он будет показан как Markdown текст. В остальных случаях, он будет показываться как преварительно отформатированный текст.

**Пропустить**  
Отметьте этот чекбокс, если текст описывает работу и возможности скрипта и не должен показываться при выполнении скрипта.
