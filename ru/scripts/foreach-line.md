# Для каждой строки

Команда **Для каждой строки** выполняет вложенные команды для каждой строки в указанном текстe. Эту команду удобно использовать, когда необходимо выполнить какие-то действия для каждой строки из определённого набора.

%html.foreach-line%

**Текст**  
Укажите исходный текст, который содержит строки для обработки.

**Имя переменной**  
Укажите имя переменной, в которую будет записана очередная строка для обработки. Вы можете использовать эту переменную во вложенных  скриптах.
