# Удалить строки

**v1.21.0**  
Команда **Удалить строки** удаляет в тексте строки, которые содержат хотя бы одну из указанных подстрок.

%html.remove-lines%

**Текст**  
Укажите исходный текст, в котором необходимо удалить строки. Вы можете указать имя переменной в виде *#varname#* или файл
*<#path#\my.log>*.

**Удалять строки, которые содержат**  
Команда удаляет строки, которые содержат хотя бы одну подстроку из указанных здесь. Подстроки для проверки
разделяются переводом строки.

**Результирующая переменная**  
Укажите имя переменной, в которую будет записан текст с удалёнными строками.
