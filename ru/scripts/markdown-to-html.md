# Markdown в HTML

**v1.15.0**  
Команда **Markdown в HTML** конвертирует Markdown текст в HTML формат.

%html.markdown-to-html%

**Имя переменной**  
Укажите имя переменной, которая содержит текст в формате Markdown.

**Результирующая переменная**  
Укажите имя переменной, в которую будет записан полученный HTML текст. Если имя переменной не указано, то результат будет присвоен исходной переменной.
