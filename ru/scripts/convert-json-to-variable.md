# Конвертировать JSON в переменную

Команда **Конвертировать JSON в переменную** преобразует текст в формате JSON в переменную объект. 

%html.convert-json-to-variable%

**Текст в формате JSON**  
Вы можете указать JSON данные или имя переменной, которая содержит JSON строку.

**Имя переменной**  
Укажите имя переменной, которой будет присвоен созданный объект.

Например, если была указана следующая JSON строка

```json
{
  "name": "John Doe",
  "Company": "My Company",
  "age": 36,
  "ids": [1,2,3]
}
```

и имя результирующей переменной *user*, то обращение к полям переменной возвратит такие значения

```txt
#user.name# => John Doe
#user.age# => 36
#user.ids[1]# => 2
#user.ids# => [1,2,3]
```
