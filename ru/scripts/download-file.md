# Скачать файл

Команда **Скачать файл** скачивает файл по указанному URL адресу и сохраняет его в указанной директории.

%html.download-file%

**URL**  
Укажите адрес файла, которой вы хотите скачать.

``` txt
https://#mydomain#/archive/myapp.zip
```

**Целевая папка**  
Укажите директорию, куда следует сохранить скаченный файл.

**Новое имя файла**  
По умолчанию, файл сохраняется с именем, которое указано в URL. Но вы можете в этом поле указать другое имя для сохраняемого файла.
