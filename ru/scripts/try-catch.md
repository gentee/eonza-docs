# Попытка / Перехват

По умолчанию, скрипт прекращает работу при возникновении ошибки. Если вы хотите повторить какие-то действия или проигнорировать возникшую ошибку, то используте команду **Попытка / Перехват**. Если ошибка возникнет в любой из вложенных команд, то вы увидите сообщение с текстом ошибки и можете выбрать один из трех вариантов:

* **Повторить** - начать повторное выполнение начиная с первого вложенного скрипта.
* **Прервать** - прервать выполнение текущего скрипта.
* **Игнорировать** - продолжить выполнение скрипта с команды, которая находится следующей за командой *Попытка / Перехват*. В этом случае, будут пропущены все вложенные команды, которые находятся ниже команды, в которой произошла ошибка.
