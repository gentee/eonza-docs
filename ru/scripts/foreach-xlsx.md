# Для каждого ряда в XLSX

Команда **Для каждого ряда в XLSX** выполняет вложенные команды для каждой ряда в указанном XLSX файле.

%html.foreach-xlsx%

**Имя файла**  
Укажите полный или относительный путь к XLSX файлу, в котором вы хотите прочитать каждый ряд и выполнить вложенные команды.

**Имя переменной**  
Укажите имя переменной, в которую будет записан массив значений очередного ряда. Нумерация начинается с нуля. Вы можете использовать эту переменную объект во вложенных командах. Например, если вы указали имя переменной *val*, то значение *#val[0]#* будет равно значению ячейки в текущем ряде в колонке *A*. *#val[1]#* будет равно значению в колонке *B* и т.д.

**Колонки**  
По умолчанию, поля каждой строки записываются в переменную массив. Но вы можете указать имя для каждой колонки. Имена разделяются запятыми. В этом случае, значения колонок будут записаны в ассоциативный массив. Если вы не укажете имя для колонки, то она не будет доступна. Например, пусть у нас указана переменная *val* и значение этого параметра равно *name,company,id*. В этом случае, *#val.name#* равно значению в колонке *A*, *#val.company#* равно значению в колонке *B*, *#val.id#* равно значению в колонке *C*. Если в этом параметре указать *name,,id*, то будут определены только значения *#val.name#* и *#val.id#*.

**Имя листа**  
По умолчанию, скрипт перебирает ряды в первом листе. Вы можете указать другое имя листа, в котором нужно
выполнить действия для каждого ряда.
