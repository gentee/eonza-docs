# Создать событие

**v1.16.0**  
Команда **Создать событие** создает [событие](/ru/docs/events.html) в программе Eonza. В этом случае, программа Eonza запустит соответствующий скрипт.

%html.create-event%

**Имя события**  
Укажите имя события, которое нужно создать.

**Прикреплённые данные**  
Вы можете указать любой текст, который будет присвоен предопределенной константе **data** в запущенном скрипте. Скрипт должен самостоятельно обрабатывать содержимое константы **data**.

``` json
{
"num": 77,
"path": "#outfile#",
"list": [ 
        { "title": "First Item", "value": 11},
        { "title": "Second Item", "value": 13}
    ]
}
```
