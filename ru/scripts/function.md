# Функция

Команда **Функция** определяет локальную функцию внутри скрипта. Эта команда может быть полезна, если нужно выполнить одну и ту же последовательность команд в разных местах скрипта. В этом случае, вы можете добавить эти команды внутрь функции и вызывать её в нужных местах. Внутри функции доступны все переменные скрипта.

%html.function%

**Имя функции**  
Укажите имя функции. Это имя будет использоваться в команде [Вызвать функцию](call-function.html).
