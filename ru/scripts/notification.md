# Уведомление

Команда **Уведомление** отправляет уведомление в программу Eonza. Полученные уведомления можно смотреть в закладке [Уведомления](/ru/docs/notifications.html).

%html.notification%

**Текст**  
Укажите текст отправляемого уведомления. Текст может содержать HTML тэги.

``` txt
Hello, #name#,
The test task has been <strong>successfully</strong> finished.
```
