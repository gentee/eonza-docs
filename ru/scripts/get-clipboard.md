# Получить буфер обмена

Команда **Получить буфер обмена** сохраняет содержимое буфера обмена в переменную.

%html.get-clipboard%

**Имя переменной**  
Укажите имя переменной, которой будет присвоено содержимое буфера обмена. Команда может получить только текст из буфера обмена.

**Очистить буфер обмена**  
Отметьте этот чекбокс, если вы хотите очистить буфер обмена после получения его содержимого.
