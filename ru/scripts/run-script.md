# Запустить скрипт

Команда **Запустить скрипт** запускает указанный скрипт как отдельный процесс и продолжает работу текущего скрипта.

%html.run-script%

**Скрипт**  
Укажите системное имя скрипта, который нужно запустить.

**Прикреплённые данные**  
Вы можете указать любой текст, который будет присвоен предопределенной константе **data** в запущенном скрипте.

``` yaml
str: Test message
val: 12345
path: /home/user/eonza
```

**Запустить скрытно**  
Отметьте этот чекбокс, если вы не хотите открывать запущенный скрипт в браузере.
