# Отправить письмо

**1.12.0**  
Команда **Отправить письмо** отправляет электронное письмо используя указанный SMTP сервер.

%html.send-email%

**SMTP сервер**  
Укажите имя переменной, которая содержит настройки SMTP сервера. Эта переменная должна быть определена с помощью команды [SMTP сервер](smtp-server.html).

**От**  
Укажите email отправителя письма. Если отправитель не указан, то будет использовано имя пользователя для подключения к SMTP серверу. Рекомендуется, чтобы отправитель совпадал с именем пользователя, в противном случае, почтовый сервер может не отправить письмо.

``` txt
eonza.dev@gmail.com
Support Team <eonza.dev@gmail.com>
```

**Кому**  
Укажите email пользователя, которому вы хотите отправить письмо.

``` txt
johndoe@eonza.org
John Doe <johndoe@eonza.org>
```

**Тема письма**  
Укажите тему письма.

**Содержимое письма**  
Укажите текст письма. Если вы хотите отправить письмо с HTML разметкой, то текст должен начинаться с тэга *&lt;html&gt;*.

``` html
<html>
<body>
Hello, <b>#name#</b>,<br>
<p>
...
</p>
Best regards,<br>
#myinfo#
</body>
</html>
```
