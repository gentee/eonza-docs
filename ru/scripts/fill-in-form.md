# Заполнить форму

Команда **Заполнить форму** присваивает указанные значения HTML элементам в соответствии с их идентификатором **id** или атрибутом **name**. Вы можете использовать эту команду в скриптах, которые запускаются в [расширении Eonza](/ru/docs/chrome-extension.html) для Chrome браузера. Если ваш скрипт использует эту команду, то запускайте его в расширении в скрытом режиме, чтобы браузер не переключался на новую закладку.

%html.silent-ext%

**URL**  
Укажите URL, где необходимо заполнить форму. Если адрес заканчивается символом '*', то формы будут заполнятся на всех веб-страницах, которые начинаются с указанной строки. В противном случае, форма заполнится только на указанной странице.

Так как, эта команда выполняется только на определенных страницах, то вы можете в одном скрипте добавить много таких команд для разных веб-страницах. В этом случае, при запуске вашего скрипта в расширении браузера, скрипт сравнит адрес текущей веб-страницы со всеми командами **Заполнить форму** и подставит значения, которые соответствуют текущему URL.

%html.fill-in-form%

## Список input элементов

Добавьте в этот список идентификаторы или имена HTML контролов, которые необходимо заполнить.

**ID или имя HTML элемента**  
Укажите имя идентификатора **id** или атрибута **name** HTML элемента в котором нужно подставить значение. Если веб-страница имеет несколько элементов с указанным атрибутом **name**, то значение подставится в первый найденный элемент.

**Значение**  
Укажите значение, которое необходимо присвоить данному HTML элементу. Если вы хотите вставлять пароли и другую секретную информацию, то рекомендуется использовать [безопасное хранилище](/ru/docs/pro-storage.html) Pro версии.
