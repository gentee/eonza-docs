# Выполнить SQL

**1.12.0**  
Команда **Выполнить SQL** выполняет SQL запрос.

%html.sql-exec%

**SQL соединение**  
Укажите имя переменной с идентификатором SQL соединения. Эта переменная должна быть определена в команде [SQL соединение](sql-connection.html).

**SQL запросы**  
Укажите SQL запрос, который необходимо выполнить. Если вы хотите указать несколько SQL запросов, то разделите их комментариями, начинающиеся с *#* или *--*.

``` sql
DROP TABLE IF EXISTS `#dbtable#`;
-- comment
CREATE TABLE #dbtable# ( id INT PRIMARY KEY AUTO_INCREMENT, 
     name VARCHAR(255), count INT, flag BOOLEAN);
```

**Параметры**  
Рекомендуется использовать параметры при использовании значений в SQL запросе. Если у вас указано несколько SQL запросов, то параметры будут применены к последнему SQL запросу.

Например, при использовании любого из этих запросов вам необходимо добавить два параметра. Вы можете использовать подстановку переменных *#varname#* в значениях параметрах.

``` sql 
INSERT INTO `#dbtable#` (name, count, flag) VALUES(?, ?, TRUE);  # mysql 
INSERT INTO "#dbtable#" (id, name, count, flag) VALUES(1, $1, $2, TRUE); # postgresql
```

``` txt
#name# (#id#)
#icount#
```
