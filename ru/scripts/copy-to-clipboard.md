# Копировать в буфер обмена

Команда **Копировать в буфер обмена** копирует указанный текст в буфер обмена.

%html.copy-to-clipboard%

**Текст**  
Укажите текст, который необходимо скопировать в буфер обмена. Если текст не указан, то буфер обмена будет очищен.

``` txt
Name: #name#
Company: #company#
```
