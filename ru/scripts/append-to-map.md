# Добавить к хэш-массиву

**v1.21.0**  
Команда **Добавить к хэш-массиву** добавляет элемент к переменной объекту. Объект должен быть ассоциативным массивом.
Если указанная переменная не существует, то она будет создана.

%html.append-to-map%

**Имя переменной**  
Укажите имя переменной, которая является ассоциативным массивом или которую вы хотите создать. Вы можете указывать
имена массивов в существующих объектах.

```txt
mymap
options.params
mymap[0].settings.sub
```

**Ключ**  
Укажите ключ, которому будет присвоено значение.

**Значение**  
Укажите добавляемое значение. Оно может быть многострочным. По умолчанию, к ассоциативному массиву будет добавлен объект с этой строкой. Также, вы можете указать имя другого объекта. Скрипт проверяет, если существует объект с указанным именем, то в этом случае, к хэш-массиву будет добавлен этот объект.

**Заменить значения переменных**  
Значения этого параметра описаны в команде [Установить переменную](set-variable.html).
