# Переместить файл

Команда **Переместить файл** перемещает указанный файл или директорию.

%html.move-file%

**Путь**  
Укажите полный или относительный путь к директории или файлу, который нужно переместить.

**Целевой каталог**  
Укажите директорию, куда нужно перенести указанный файл или директорию.

**Если файл существует**  
Данный параметр описан в команде [Копировать файл](copy-file.html). Этот параметр не используется при перемещении директории.

## Опциональные параметры

Вы можете использовать такие же опциональные параметры как в команде **Копировать файл**.
