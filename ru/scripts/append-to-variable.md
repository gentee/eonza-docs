# Добавить к переменной

Команда **Добавить к переменной** добавляет текст к значению переменной. Если указанная переменная не существует, то она будет создана.

%html.append-to-variable%

**Имя переменной**  
Укажите имя переменной, к которой вы хотите добавить текст.

**Разделитель**  
Можно указать разделитель, который будет добавлен перед добавляемым текстом. Разделитель не добавляется, если переменная не существует, она равна пустой строке или текущее значение заканчивается этим разделителем.

```txt
#.r##.n#  - это разделитель \r\n (0xD 0xA).
#.s# - разделитель пробел.
```

**Значение**  
Укажите добавляемое значение. Оно может быть многострочным. 

**Заменить значения переменных**  
Значения этого параметра описаны в команде [Установить переменную](set-variable.html).
