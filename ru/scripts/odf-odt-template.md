# Шаблон Odf/Odt

Команда **Шаблон Odf/Odt** заменяет переменные в Odt шаблоне (Libre Writer). Так как скрипт сам производит все замены в документе, то ему не требуется установленного Libre Office. Если команда не заменяет какие-то переменные в шаблоне, то нужно отдельно скопировать их (*#varname#*) из обычного текстового редактора и вставить в документ. Это связано с тем, что
Libre Writer может разбить имя переменной на невидимые фрагменты, которые скрипт не может обработать как одно целое.

%html.odf-odt-template%

**Шаблон (.odt)**  
Укажите *odt* файл, которые содержит переменные вида *#varname#*. Данный скрипт заменит все переменные и
сохранит новый документ с указанным ниже именем.

**Файл вывода**  
Укажите полный или относительный путь к документу, который нужно создать. Если файл существует, то он будет перезаписан.
