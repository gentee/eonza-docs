# Получить INI значения

Команда **Получить INI значения** получает одно или несколько значений из указанной секции INI файла.

%html.read-ini-values%

**Имя файла**  
Полное или относительное имя INI файла.

**Секция**  
Имя секции.

## Ключи

Укажите ключи, значения которых вы хотите прочитать.

**Имя ключа**  
Имя ключа, который будет получен.

**Имя переменной**  
Имя переменной, которой будет присвоено значение.

**Значение по умолчанию**  
По умолчанию, если указанного ключа не существует, то возвращается ошибка. Если вы укажете параметр *Значение по умолчанию*, то в этом случае ошибки не будет, а переменной будет присвоена указанная строка.
