# Установить значение Реестра

Команда **Установить значение Реестра** создает или изменяет значения Реестра Windows в указанном подключе.

%html.set-registry-value%

**Корневой ключ**  
Укажите корневой ключ Реестра.

**Подключ**  
Укажите имя ключа Реестра в котором находятся устанавливаемые значения.

``` txt
Software\My Application
```

## Значения
Укажите здесь параметры ключа, которые необходимо создать или изменить.

**Имя значения**  
Имя значения, которое будет создано или изменено. Если указана пустая строка, то будет изменено значение по умолчанию.

**Тип значения**  
Укажите тип значения. Текущая версия поддерживает следующие типы значений Реестра: **SZ, EXPAND_SZ, DWORD**.

**Значение**  
Укажите строку, которая будет присвоена данному значению.

**Доступ**  
Доступ для ключей 32-битных или 64-битных приложений.

* *---* - доступ по умолчанию.
* *WOW64_32KEY* - доступ к ключам Реестра для 32-битных приложений.
* *WOW64_64KEY* - доступ к ключам Реестра для 64-битных приложений.
