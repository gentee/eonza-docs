# Размер директории

Команда **Размер директории** получает размер и количество файлов в указанной директории. Подсчёт ведется по всем файлам в поддиректориях.

%html.directory-size%

**Путь**  
Укажите полный или относительный путь к директории, информацию о которой вы хотите получить.

**Результирующая переменная**  
Укажите имя переменной куда будет записан суммарный размер файлов в директории. Кроме этого, будут создана переменная объект с таким же именем со следующими полями:

* **size** - размер всех файлов в директории.
* **files** - количество файлов в директории.
* **dirs** - количество поддиректорий в директории.

Например, если вы указали имя переменной *di*, то вы можете получить подобные значения:

``` txt
#di# = "134594040"
#di.size# = "134594040"
#di.files# = "253"
#di.dirs# = "12"
```
