# Удалить значение Реестра

**v1.14.0**  
Команда **Удалить значение Реестра** удаляет значения Реестра Windows в указанном подключе.

%html.delete-registry-value%

**Корневой ключ**  
Укажите корневой ключ Реестра.

**Подключ**  
Укажите имя ключа Реестра в котором нужно удалить значение.

``` txt
Software\My Application\Data
```

## Значения для удаления
Вы можете указать одно или несколько значений, которые вы хотите удалить.

**Имя значения**  
Имя значения, которое будет удалено. Если указана пустая строка, то будет удалено значение по умолчанию.

**Доступ**  
Доступ для ключей 32-битных или 64-битных приложений.

* *---* - доступ по умолчанию.
* *WOW64_32KEY* - доступ к ключам Реестра для 32-битных приложений.
* *WOW64_64KEY* - доступ к ключам Реестра для 64-битных приложений.
