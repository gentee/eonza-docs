# Переименовать файлы

**v1.10.0**  
Команда **Переименовать файлы** переименовывает файлы в соответствии с указанным регулярным выражением.

%html.rename-files%

**Путь**  
Укажите директорию с файлами, которые должны быть переименованы.

**Регулярное выражение**  
Укажите регулярное выражение, которое будет использоваться для переименования файлов. Файлы, которые не совпадают с данным регулярным выражение, будут пропущены.

**Новое имя файла**  
Укажите новое имя файла. Необходимо указать только ту часть, которая будет заменёна на найденный фрагмент для указанного регулярного выражения. В этом параметре можно указывать *$i* или *${i}* для подстановки i-го подсовпадения.

``` txt
Original Name: myfile (1).txt
Regular Expression: \s\(\d\)\.
New Filename: .
Result filename: myfile.txt

Original Name: myfile (1).txt
Regular Expression: \((\d+)\)
New Filename: [${1}]
Result filename: myfile [1].txt
```

**Если файл существует**  
Вы можете указать действие для случая, когда файл с новым именем существует. Варианты описаны в команде [Копировать файл](copy-file.html).

**Рекурсивный поиск**  
**Исключать по маске или рег.выраж.**  
Эти параметры описаны в команде [Цикл для каждого файла](foreach-file.html).
