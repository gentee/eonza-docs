# Шаблон XLSX

**v1.21.0**  
Команда **Шаблон XLSX** заменяет переменные в Xlsx шаблоне (Microsoft Excel). Так как скрипт сам производит все замены в электронной таблице, то ему не требуется установленного Microsoft Office. Замена производится на всех листах электронной таблицы.

%html.xlsx-template%

**Шаблон (.xlsx)**  
Укажите *xlsx* файл, которые содержит переменные вида *#varname#*. Данный скрипт заменит все переменные и
сохранит новую электронную таблицу с указанным ниже именем.

**Файл вывода**  
Укажите полный или относительный путь к документу, который нужно создать. Если файл существует, то он будет перезаписан.
