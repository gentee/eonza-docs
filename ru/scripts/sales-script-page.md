# Страница скрипта продаж

Команда **Страница скрипта продаж** создаёт страницу для скрипта продаж. Вы можете указать необходимый текст и варианты ответов для переходов на соответствующие страницы. Эта команда не производит никаких действий. Чтобы запустить скрипта продаж нужно использовать команду [Скрипт продаж](sales-script.html).

%html.sales-script-page%

**Имя**  
Укажите имя страницы. Используйте это имя в вариантах ответов других страниц. При выборе такого варианта, пользователь перейдёт на эту страницу.

**Название**  
Заголовок страницы. Если не указан, то заголовок будет равен имени страницы.

**Текст**  
Текст страницы. Вы можете использовать текст в формате HTML или Markdown. 

``` html
Hello, <span style="color: #00f"><b>world</b></span>!
```

## Варианты ответов

**Текст**  
Укажите текст ссылки для варианта ответа. Если не указан, то будет использоваться заголовок страницы скрипта продаж, которая определена в следующем поле.

**Страница скрипта продаж**  
Укажите имя страницы скрипта продаж, которое откроется при выборе данного варианта ответа.
