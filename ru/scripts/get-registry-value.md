# Получить значение Реестра

Команда **Получить значение Реестра** получает значения Реестра Windows в указанном подключе и присваивает эти значения переменным.

%html.get-registry-value%

**Корневой ключ**  
Укажите корневой ключ Реестра.

**Подключ**  
Укажите имя ключа Реестра в котором находятся получаемые значения.

``` txt
Software\My Application
```

## Значения
Укажите здесь значения, которые необходимо получить. Текущая версия поддерживает следующие типы значений Реестра: **SZ, EXPAND_SZ, DWORD**.

**Имя значения**  
Имя значения, которое будет получено. Если указана пустая строка, то будет получено значение по умолчанию.

**Имя переменной**  
Имя переменной, которое будет присвоено значение.

**Значение по умолчанию**  
По умолчанию, если значения не существует, то возвращается ошибка. Если вы укажете параметр *Значение по умолчанию*, то в этом случае ошибки не будет, а переменной будет присвоена указанная строка.

**Доступ**  
Доступ для ключей 32-битных или 64-битных приложений.

* *---* - доступ по умолчанию.
* *WOW64_32KEY* - доступ к ключам Реестра для 32-битных приложений.
* *WOW64_64KEY* - доступ к ключам Реестра для 64-битных приложений.
