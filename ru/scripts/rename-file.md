# Переименовать файл

Команда **Переименовать файл** переименовывает указанный файл или директорию.

%html.rename-file%

**Путь**  
Укажите полный или относительный путь к директории или файлу, который нужно переименовать.

**Новое имя файла**  
Укажите новое имя файла или директории.

**Если файл существует**  
Данный параметр описан в команде [Копировать файл](copy-file.html). 

## Опциональные параметры

Вы можете использовать такие же опциональные параметры как в команде **Копировать файл**.
