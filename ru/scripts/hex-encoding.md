# Hex кодирование

**1.13.0**  
Команда **Hex кодирование** кодирует байты в шестнадцатеричную строку или декодирует шестнадцатеричную строку в последовательность байт.

%html.hex-encoding%

**Имя переменной**  
Укажите имя переменной, которая содержит исходную строку.

**Кодирование**  
Укажите тип конвертации.

* *Из байт в шестнадцатеричную строку*
* *Из шестнадцатеричной строки в байты*

**Результирующая переменная**  
Укажите имя переменной, которой будет присвоен полученный результат. Если имя переменной не указано, то результирующая строка будет присвоена исходной переменной.
