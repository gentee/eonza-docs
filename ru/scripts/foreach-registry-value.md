# Для каждого значения Реестра

**v1.14.0**  
Команда **Для каждого значения Реестра** выполняет вложенные команды для каждого значения в указанном подключе Реестра Windows.

%html.foreach-registry-value%

**Корневой ключ**  
Укажите корневой ключ Реестра.

**Подключ**  
Укажите имя подключа Реестра в котором нужно перебрать все значения.

``` txt
Software\My Application
```

**Имя переменной**  
Укажите имя переменной, в которую будет записанo имя очередного значения. Вы можете использовать эту переменную во вложенных командах. Если для подключа указано значение по умолчанию, то переменная в этом случае будет равна пустой строке.

**Доступ**  
Доступ для ключей 32-битных или 64-битных приложений.

* *---* - доступ по умолчанию.
* *WOW64_32KEY* - доступ к ключам Реестра для 32-битных приложений.
* *WOW64_64KEY* - доступ к ключам Реестра для 64-битных приложений.
