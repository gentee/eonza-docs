# Вызвать функцию

Команда **Вызвать функцию** вызывает функцию, которая была определена с помощью команды [Функция](function.html).

%html.call-function%

**Имя функции**  
Укажите имя функцию, которая будет вызвана.

## Параметры

Вы можете присвоить значения переменным, которые используются в вызываемой функции.

**Имя переменной**  
Укажите имя переменной, которой вы хотите присвоить значение.

**Значение**  
Укажите присваиваемое значение. Оно может быть многострочным.
