# Создать архив

Команда **Создать архив** создает zip или tar.gz архив.

%html.create-archive%

**Файл вывода**  
Укажите путь и имя создаваемого файла архива. У файла укажите соответствующее расширение.

**Тип**  
Выберите тип создаваемого архива *zip* или *tar.gz*.

**Путь**  
Укажите директорию, которую нужно упаковать в архив. По умолчанию, будут сжиматься все файлы и поддиректории.

Вы можете указать шаблоны для файлов, которые необходимо упаковывать и для файлов и директорий, которые нужно пропускать.

**Рекурсивныйй поиск**  
**Маска или регулярное выражение**  
**Исключать по маске или рег.выраж.**  
Эти параметры описаны в команде [Цикл для каждого файла](foreach-file.html).
