# SQL запрос

**1.12.0**  
Команда **SQL запрос** выполняет SQL запрос для выбора данных и записывает полученные данные в переменную объект.

%html.sql-query%

**SQL соединение**  
Укажите имя переменной с идентификатором SQL соединения. Эта переменная должна быть определена в команде [SQL соединение](sql-connection.html).

**Запрос SELECT**  
Укажите SQL запрос SELECT для получения записей из базы данных.

``` sql
select * from #dbtable# order by id
```

**Параметры**  
Рекомендуется использовать параметры при использовании значений в SQL запросе. 

Например, при использовании этого запроса вам необходимо добавить один параметр. Вы можете использовать подстановку переменных *#varname#* в значениях параметрах.

``` sql
select * from #dbtable# where iduser = ? order by id
```

``` txt
#iduser#
```

**Результирующая переменная**  
Укажите имя переменной, в которую будет сохранён полученный массив записей. Переменная является объектом, в котором каждый элемент массива является ассоциативным массивом со значениями колонок. В дальнейшем, вы можете использовать команду [Для каждого](foreach.html) для перебора всех полученных записей или подставлять конкретные значения.

``` txt
#list[0].name#
#mylist[#index#].count#
```
