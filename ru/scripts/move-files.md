# Переместить файлы

**v1.10.0**  
Команда **Переместить файлы** перемещает файлы из одной директории в другую в соответствии с указанными масками и регулярными выражениями. Если маски и регулярные выражения не указаны, то будут перемещаться все файлы.

%html.move-files%

**Путь**  
Укажите директорию с файлами, которые должны быть перемещены.

**Целевая папка**  
Директория, куда будут перемещаться файлы. При перемещении сохраняется иерархия поддиректорий.

**Если файл существует**  
Вы можете указать действие для случая, когда перемещаемый файл существует в целевой директории. Варианты описаны в команде [Копировать файл](copy-file.html).
