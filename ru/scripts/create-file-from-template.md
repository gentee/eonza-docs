# Создать файл из шаблона

Команда **Создать файл из шаблона** создает файл из текстового шаблона.

%html.create-file-from-template%

**Шаблон**  
Укажите шаблон из которого нужно создать файл. При записи в файл в шаблоне будут заменены все переменные на их текущие значения. Вы можете хранить шаблон в виде файла. В этом случае, укажите путь к файлу-шаблону в угловых скобках.

``` txt
User: #user#
Group: #usergroup#
Status: #status#
```

**Файл вывода**  
Укажите полный или относительный путь к файлу, который нужно создать. Если файл существует, то он будет перезаписан.
