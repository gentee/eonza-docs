# Разобрать YAML

Команда **Разобрать YAML** разбирает YAML данные и сохраняет их в переменную объект.

%html.parse-yaml%

**YAML данные**  
Указажите YAML данные, которые необходимо разобрать. Если у вам нужно разобрать yaml файл, то укажите его имя в угловых скобках или предварительно прочитайте его в переменную. Ниже приведены несколько вариантов определений данного параметра.

``` yaml
name: Alex
list:
    - id: 123
      value: test
    - id: 56
      value: "#value# #mark#"
```

``` txt
#yamlvar#
```

``` txt
</home/user/data/settings.yaml>
```

**Результирующая переменная**  
Укажите имя переменной, которой будет присвоен объект с YAML данными. В дальнейшем, вы можете использовать эту переменную в командах и функциях для работы с объектами. Также, можно непосредственно получать значения полей. Например, если вы указали имя переменной *yamlobj*, то для первого примера можно получить значения полей следующим образом:

``` txt
#yamlobj.name# => Alex
#yamlobj.list[0].value# => test
#yamlobj.list[1].id# => 56
```
