# Зашифровать файл

Команда **Зашифровать файл** зашифровывает указанный файл и сохраняет зашифрованные данные как новый файл. Этот файл может быть расшифрован с помощью команды [Расшифровать файл](decrypt-file.html).

%html.encrypt-file%

**Имя файла**  
Укажите полный или относительный путь к файлу, который нужно зашифровать. Исходный файл не будет изменён.

**Алгоритм**  
Выберите алгоритм шифрования.

* *AES-256*

**Пароль**  
Укажите пароль шифрования. **Не указывайте в скриптах пароли в явном виде!**

**Целевая папка**  
Директория, куда будет сохранён зашифрованный файл. Если она не указана, то файл будет сохранён в папку исходного файла.

**Файл вывода**  
Имя зашифрованного файла. Если не указано, то к имени исходного файла будет добавлено расширение *.ec*.
