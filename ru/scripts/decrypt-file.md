# Расшифровать файл

Команда **Расшифровать файл** расшифровывает указанный файл. Этот файл должен быть зашифрован с помощью команды [Зашифровать файл](encrypt-file.html).

%html.decrypt-file%

**Имя файла**  
Укажите полный или относительный путь к файлу, который нужно расшифровать. Исходный файл не будет изменён.

**Пароль**  
Укажите пароль шифрования. **Не указывайте в скриптах пароли в явном виде!**

**Целевая папка**  
Директория, куда будет сохранён расшифрованный файл. Если она не указана, то файл будет сохранён в папку исходного файла.

**Файл вывода**  
Имя расшифрованного файла. Если не указано, то имя будет такое же как у оригинального файла, который был зашифрован.
