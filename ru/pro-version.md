# PRO Версия

Для развития бесплатной open-source программы необходимы финансовые ресурсы, поэтому в наших планах стоит разработка и выпуск PRO версии. Она будет иметь возможности, которые сделают работу с программой более удобной, но не затронут основной функционал. Сразу отметим следующие моменты:

* Все скрипты будут работать в любой версии.
* Pro версия будет продаваться в виде годовой лицензии, которая будет распространяться на все обновления в течении года.
* Ориентировочное время выхода Pro версии - апрель-май 2021.
* Обновления бесплатной версии будут по-прежнему выходить 1-2 раза в месяц.
