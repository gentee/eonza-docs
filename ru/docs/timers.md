# Таймеры

**v1.15.0**  
Таймеры позволяют вам установить автоматический запуск скриптов с определенной периодичностью. Минимальный интервал между срабатыванием таймера равен одной минуте. Если вы хотите выполнять какие-то действия чаще, то тогда нужно создать скрипт с  бесконечным циклом и [паузами](/ru/scripts/sleep.html) в необходимое количество секунд. Таймеры работают только тогда, когда программа Eonza запущена.

%html.timers%

Список созданных таймеров отсортирован по ближайшему времени запуска. У каждого таймера в таблице указывается имя, периодичность запуска в формате **Cron** и имя запускаемого скрипта.

## Параметры таймера

%html.timer-cron%

* **Имя**  
Имя таймера.

* **Скрипт**  
Системное имя скрипта. Скрипт не должен иметь параметров, так как в противном случае, он будет ожидать ввод параметров при запуске.

* **Отключён**  
Отметьте этот чекбокс, если вы хотите остановить таймер.

В полях ниже, вы можете указать конкретные значения или диапазоны значений для срабатывания таймера. Значения разделяются запятыми. В правом столбце можно указать соответствующие интервалы.

* **Минута/Каждые [?] минут** - возможные значения 0-59.
* **Час/Каждые [?] часов** - возможные значения 0-23.
* **День/Каждые [?] дней** - возможные значения 1-31.
* **Месяц/Каждые [?] месяцев** - возможные значения 1-12.
* **Неделя/Каждые [?] недель** - возможные значения 0-6. Неделя начинается с воскресенья.

Например, пусть указаны следующие параметры

``` txt
Минута: 0 / Каждые 15 минут
Час: 9-17,21-23
День: 1 / Каждые 2 дня
```

Это означает, что таймер будет запускать скрипт по нечетным дням каждые 15 минут (0, 15, 30, 45) с 9 до 17 и с 21 до 23 часов включительно.
