# Расширение для Chrome

Вы можете установить и использовать расширение Eonza для браузера Chrome. Это расширение позволяет запускать скрипты, которые были определены в закладке [Браузер](browser.html). При этом, при запуске скрипта ему передается URL адрес и заголовок текущей страницы в браузере. Также, скрипту может быть передано содержимое HTML страницы. Расширение будет работать в любом браузере, который поддерживает расширения Chrome. Рассмотрим, как установить и настроить это расширение браузера.

## Установка расширения

Вы можете самостоятельно скачать и собрать расширение из исходников, которые расположены в [репозитории на github.com](https://github.com/gentee/eonza-chrome). Самый простой путь - [**скачать готовый архив**](https://github.com/gentee/eonza-chrome/releases/download/v1.0.0/eonza-chrome.zip) и распаковать его в любую директорию.

После того, как файлы расширения были распакованы, запустите браузер и откройте страницу с расширениями. Это можно сделать в разделе настроек или указав *chrome://extensions/* в адресной строке. Включите **Режим разработчика** и нажмите на кнопку **Загрузить распакованное расширение**. В окне выбора директории укажите директорию с распакованными файлами.

%html.chrome-ext-install%

После того, как расширение установлено, нажмите на иконку расширений и затем на кнопку *Прикрепить*, чтобы иконка расширения постоянно показывалась на панели с адресной строкой.

%html.chrome-ext%

## Настройка расширения

После того как расширение установлено, нажмите на его иконку и должно появиться окно с кнопками *Настройки* и *Помощь*.

%html.chrome-ext-start%

Нажмите на кнопку *Настройки*, чтобы указать адрес приложения Eonza.

%html.chrome-ext-settings%

Сейчас необходимо добавить скрипты в [закладку Браузер](browser.html) в программе Eonza. Для примера, можете добавить скрипт *url.title* (Copy URL & Title), который будет копировать в буфер обмена URL и заголовок текущей страницы в браузере.
Если все настройки указаны правильно и программа Eonza запущена, то вы должны увидеть список скриптов в расширении. Следует заметить, что если Eonza защищена паролем, то необходимо открыть её в браузере в отдельной закладке. В этом случае, аутентификация будет происходить автоматически.

%html.chrome-ext-run%

Для каждого скрипта показывается две кнопки. При нажатии на большую кнопку с именем скрипта, он будет запущен и открыт в браузере. При нажатии на правую кнопку со стрелкой, скрипт будет запущен в скрытом режиме. При запуске, скрипту в константе *#.data#* передаётся JSON строка с URL и заголовком текущей страницы. Также, может быть передано HTML содержимое страницы для её парсинга.
