# Пакеты

**v1.23**  
Пакет - это набор скриптов, которые предназначены для работы с определённой внешней программой или сервисом. Также пакет может объединять скрипты, которые реализуют действия с одними и теми же объектами. Каждый пакет может иметь несколько параметров, которые определяются в диалоговом окне *Настройки*. Эти параметры могут использоваться в любом скрипте пакета. 

По умолчанию в программе Eonza доступны только скрипты стандартной библиотеки. Если вам нужны скрипты определённого пакета, то достаточно установить его. В этом случае, в закладке *Скрипты* появится папка с именем пакета и содержащая его скрипты. Скрипты пакета защищены от изменения и удаления. Если вы перестали пользоваться возможностями какого-то пакета, то можете удалить его. В этом случае, папка пакета удалиться из общего списка скриптов. Настройки пакета будут сохранены и восстановятся, если пакет будет снова установлен в дальнейшем.
