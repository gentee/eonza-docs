# Автозаполнение форм

**v1.25**  
Pro версия программы Eonza даёт возможность сохранять значения полей, которые вы указывайте в формах при выполнении скриптов. Поля для паролей не сохраняются. Чтобы сохранить поля формы отметьте чекбокс **Сохранить данные формы**, который отображается ниже всех элементов формы. 

%html.autofill-forms-1%

При следующем запуске скрипта в поля формы будут подставлены предыдущие значения. Данные форм сохраняются отдельно для каждого пользователя. Программа можете запомнить несколько вариантов значений в форме. Для этого откройте дополнительные поля и укажите там имя для текущего набора значений, которые нужно запомнить. 

%html.autofill-forms-2%

В дальнейшем, вы можете выбрать имя набора значений в выпадающем списке и значения полей будут подставлены в форму.

Формы с одними и теми же полями в разных скриптах считаются разными и они отдельно хранят сохранённые наборы значений. Если вы хотите, что одинаковые формы считались разными в одном скрипте, то определите для них параметр **ref** в [общих настройках команды](common-settings.html).
