# Уведомления

Закладка Уведомления содержит список полученных уведомлений. Уведомления могут быть созданы самой программой или командой [Уведомление](/ru/scripts/notification.html). В первом случае, это могут быть уведомления о наличии новой версии или системные сообщения о работоспособности программы. Во втором случае, уведомление с любым текстом может быть отправлено из любого скрипта с помощью указанной команды. Программа показывает только 25 последних уведомлений. Все уведомления хранятся в файле с расширением *eon*. Если у вас накопилось много уведомлений и вы не хотите удалять их по одному, то можете удалить этот файл и перезапустить программу.
