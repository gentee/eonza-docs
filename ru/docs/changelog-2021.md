# История изменений за 2021

<!--## Бета версия 1.8.0

## Версия 1.11.0 - 2021/01/22

* Добавлена командa [Копировать в буфер обмена](/ru/scripts/copy-to-clipboard.html).
* Добавлена командa [Получить буфер обмена](/ru/scripts/get-clipboard.html).
* Добавлены [Уведомления](/ru/docs/notifications.html).
* Добавлена командa [Уведомление](/ru/scripts/notification.html).
* Добавлена командa [Отправить сообщение в Telegram](/ru/scripts/send-telegram-message.html).
* Добавлен выбор языка при первом запуске.
* Добавлена подстановка переменных в команду **HTTP запрос**.
* Добавлена кнопка *Проверить обновление* в **Помощь - О программе**.
* Добавлена настройка для автоматической проверки обновлений в **Настройки - Общие**.

## Версия 1.10.0 - 2021/01/06

* Добавлена командa [Переименовать файлы](/ru/scripts/rename-files.html).
* Добавлена командa [Переместить файлы](/ru/scripts/move-files.html).
* Добавлена командa [Скачать файл](/ru/scripts/download-file.html).
* Добавлена командa [Для каждой строки в файле](/ru/scripts/foreach-line-file.html).
* Добавлена функция Хэш в команду **Установить переменную**.
* Добавлен параметр *Если файл существует* в команды **Перенести файл** и **Переименовать файл**.
* Добавлено *Сохранить копию* в параметр *Если файл существует* в команду **Копировать файл**.
* Добавлена кнопка *Сохранить копию* в команду **Подтверждение файла**.
* Исправлено несколько мелких ошибок.
